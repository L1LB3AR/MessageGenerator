Message Generator. Based on Motivational and Atlethes phrases. 

---------------------------------------------------------------------------------------------------------------------------------------------------

In this project we're built a generator of random messages. Using basic JavaScript and Node. I guided this project through the gym-motivational mentality
because it helped me during my past and still helping me nowadays. 

---------------------------------------------------------------------------------------------------------------------------------------------------

To activate the script, just click at the "main" file. The script welcome us and will ask us if we want to see the phrase. We can answer Y (yes) or N (no).

---------------------------------------------------------------------------------------------------------------------------------------------------

We used JavaScript, Node.js and VisualStudio Code to code this project.
